## Git Commit 규칙
- 커밋 시 작성자(author) 또는 `Co-Authored-By`에 **Claude를 절대 포함하지 않는다**.
- `Co-Authored-By: Claude ...` 라인 추가 금지.
- 커밋 메시지 본문에도 Claude 생성 표시(예: "Generated with Claude")를 넣지 않는다.
