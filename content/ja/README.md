# Tech Bridge Book

Tech Bridge Bookは、非開発者がAIと実用的なテクノロジーを基礎から仕事での活用まで学ぶためのGitBook文書です。

この初期構造は、完全なレッスン集ではありません。レッスンを書く前に、学習の流れ、言語別フォルダー、用語管理、安全基準、作成テンプレートを整えます。

## 読む前の事前準備

この本は非開発者の学習者のために書かれています。速い答えを覚えることではなく、自分で判断する力を育てることが目的です。1章あたり10〜20分ほどの集中時間を見込んでください。

- ウェブブラウザ、文書エディター、表計算ソフトを基本レベルで使えること。
- 文書作成、報告、スケジュール、顧客対応など、職場の現実的な場面を思い浮かべられること。
- 会社の機密、顧客情報、個人情報をそのままAIツールに貼り付けないこと。詳しい基準は[AI安全とプライバシー](ai-safety-privacy.md)を先に読んでください。
- 答えを暗記しようとせず、「なぜそうなるのか」を問う習慣を持つこと。

準備ができたら、[紹介](introduction.md)ページから学習を始めてください。
