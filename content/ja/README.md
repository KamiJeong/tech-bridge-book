# Tech Bridge Book

Tech Bridge Bookは、非開発者がAIと実用的なテクノロジーを基礎から仕事での活用まで学ぶためのGitBook文書です。

この初期構造は、完全なレッスン集ではありません。レッスンを書く前に、学習の流れ、言語別フォルダー、用語管理、安全基準、作成テンプレートを整えます。

2. [カリキュラム概要](curriculum-overview.md)
3. [学習者ガイド](learner-guide.md)
4. [AI安全とプライバシー](ai-safety-privacy.md)
5. [章構成](chapters/README.md)

## 作成者向け基準

新しいレッスンは[テンプレート](templates/README.md)を使います。韓国語を基準原文とし、英語と日本語は意味と構造を保ちながら自然にローカライズします。
