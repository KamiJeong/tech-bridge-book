# Claude安全チェックリスト

Use this checklist before sharing information with Claude and before using Claude output in real work.

## Checklist

- Claudeの回答をそのまま信じず、重要な内容は確認する。
- パスワード、APIキー、認証情報、アクセストークンを貼り付けない。
- 顧客の個人情報、識別番号、連絡先、決済情報を共有しない。
- 会社の機密情報、内部戦略、非公開文書を組織の許可なく共有しない。
- 法律、医療、金融、安全に関する判断の唯一の根拠にしない。
- 最新情報だと決めつけず、日付と出典を確認する。
- 人の確認なしにClaudeの出力を最終成果物として提出しない。
- 考えることを避けるためではなく、よりよく考えるために使う。
- 自分で検証できないタスクでは利用範囲を狭めるか使わない。
- ファイルをアップロードする前に権限、保存方針、共有ルールを確認する。

## File Upload Rule

機能の利用可否はClaudeのプランや現在の画面によって変わる場合があります。自分のClaude画面で確認してください。

実務で使う前に、人が事実、文脈、表現、機密情報の有無を確認する必要があります。

## Lesson Integration

Every lesson must include at least two safety items relevant to that lesson's scenario.
