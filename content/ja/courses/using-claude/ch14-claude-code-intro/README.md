# 上級ルートとしてのClaude Code入門

この文書はGitBookでClaudeコースを探索し執筆するための基準です。

## 目標

Claude Codeがどのような場面で関係するかを理解します。

## 学習成果

ClaudeチャットとClaude Codeの開発ワークフローの違いを説明できます。

## 重要概念

一般チャットではなく開発ワークフローの道具としてのClaude Code。

## 実務での使い方

初心者開発者や自動化学習者にClaude Codeが合うか判断する。

## レッスン作成プレースホルダー

このチャプターは現在、コース構造のみを定義しています。完全なレッスンを書くときはコースのレッスンテンプレートを使い、すべての必須セクションを含めてください。

- タイトル
- Who this lesson is for / このレッスンの対象
- 学ぶこと
- 実務で重要な理由
- Core concept explanation / 重要概念の説明
- Simple example / やさしい例
- Claudeプロンプト例
- 実践演習
- 考える質問
- よくある間違い
- 安全チェックリスト
- まとめ
- 次のステップ

## 安全メモ

機能の利用可否はClaudeのプランや現在の画面によって変わる場合があります。自分のClaude画面で確認してください。

実務で使う前に、人が事実、文脈、表現、機密情報の有無を確認する必要があります。
