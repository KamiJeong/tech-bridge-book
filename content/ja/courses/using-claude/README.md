# Claude の使い方

考え、問い、確認し、磨き上げる実務向けAI協働コース

このコースはClaudeの機能一覧ではありません。Claudeと一緒に考え、質問し、検証し、成果物を改善する習慣を学びます。

## Target Audience

非開発者、初心者開発者、企画担当者、プロダクトマネージャー、創業者、マーケター、運用担当者、デザイナー、研究者、カスタマーサポート、QA担当者.

## What Learners Will Be Able To Do

- Claudeとは何か、チャットボット、文章作成パートナー、思考パートナー、調査補助、文書補助、コーディング支援、Claude Codeの違いを説明する。
- 長文要約、メール改善、議事録整理、提案書の下書き、表やチェックリスト作成などの実務用途を選べる。
- Claudeを使うべきでない場面と、人による検証が必要な場面を区別する。
- 文脈、目的、形式、制約を含む明確なプロンプトを書く。
- Claudeの出力を正確性、安全性、品質の観点で確認し、最終成果物に整える。

## Multilingual Strategy

韓国語が基準原稿です。英語と日本語は同じ構造と意味を保ちながら自然にローカライズします。

## Safety Baseline

機能の利用可否はClaudeのプランや現在の画面によって変わる場合があります。自分のClaude画面で確認してください。

実務で使う前に、人が事実、文脈、表現、機密情報の有無を確認する必要があります。
