# Exercise Template: Claude の使い方

演習は特別な開発環境なしですぐ試せる内容にし、通常15分以内で終えられる量にします。

## Exercise Frame

- **Scenario**: [Fictional workplace situation]
- **Learner role**: [Persona]
- **Task**: [What the learner should do]
- **Claude prompt starter**: [Copy-ready prompt]
- **Expected output**: [What a useful answer should include]
- **Review step**: [How the learner checks the result]
- **Safety note**: [What not to share]

## Recommended Exercise Types

- Rewrite a weak prompt into a stronger prompt
- Summarize sample meeting notes
- Turn messy notes into a table
- Compare two Claude answers
- Identify missing context in a prompt
- Create a checklist from a work scenario
- Write an email in three tone variations
- Explain a technical concept to a non-developer
- Review Claude output for possible errors
- Decide what information should not be shared
- Create a small work plan
- Convert a business problem into clear instructions

## Completion Check

- The learner can complete the exercise without developer setup.
- The task creates a visible output such as a revised prompt, table, checklist, or short plan.
- The learner must review the output before treating it as usable.
