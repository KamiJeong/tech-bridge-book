# Claudeコース用語集

韓国語の用語が基準です。英語と日本語は同じ用語リストを保ちながら自然に説明します。

| Term | Language | Definition |
|---|---|---|
| AIアシスタント | Japanese | 思考、文章作成、分析、計画を支援する指示ベースのソフトウェアです。 |
| Claude | Japanese | 文章作成、文書分析、コーディング支援、構造化された思考を助けるAnthropicのAIアシスタントです。 |
| プロンプト | Japanese | Claudeに渡す質問や指示です。 |
| 文脈 | Japanese | Claudeがよい回答をするために必要な背景情報です。 |
| ハルシネーション | Japanese | もっともらしく見えても間違っている、または根拠が弱いAIの回答です。 |
| 検証 | Japanese | Claudeの出力を信頼できる情報源や人の知識で確認することです。 |
| Artifact | Japanese | 利用できる場合、文書、コード、表、インタラクティブな下書きを扱うClaudeの出力ワークスペースです。 |
| Project | Japanese | 利用できる場合、関連する会話、文脈、指示をまとめて管理するClaude機能です。 |
| セッション | Japanese | Claudeとの1つの会話または作業の流れです。 |
| トークン | Japanese | AIモデルが処理する小さなテキスト単位です。 |
| 出力 | Japanese | Claudeが生成する回答やコンテンツです。 |
| 入力 | Japanese | Claudeに提供するテキスト、ファイル、指示です。 |
| ワークフロー | Japanese | 繰り返し使える作業手順です。 |
| 安全チェックリスト | Japanese | プライバシー、正確性、責任ある利用を確認する短いレビューリストです。 |
| ローカライズ | Japanese | 意味、トーン、例を別の言語や文化に合わせることです。 |
