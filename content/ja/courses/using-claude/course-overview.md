# Course Overview: Claude の使い方

このコースはClaudeの機能一覧ではありません。Claudeと一緒に考え、質問し、検証し、成果物を改善する習慣を学びます。

## Learning Outcomes

- Claudeとは何か、チャットボット、文章作成パートナー、思考パートナー、調査補助、文書補助、コーディング支援、Claude Codeの違いを説明する。
- 長文要約、メール改善、議事録整理、提案書の下書き、表やチェックリスト作成などの実務用途を選べる。
- Claudeを使うべきでない場面と、人による検証が必要な場面を区別する。
- 文脈、目的、形式、制約を含む明確なプロンプトを書く。
- Claudeの出力を正確性、安全性、品質の観点で確認し、最終成果物に整える。

## Prerequisite Knowledge

- Can use a web browser and type comfortably.
- Has access to Claude or a Claude-like interface approved by their organization.
- No programming knowledge is required for the main course.
- Claude Code appears only as an optional advanced path.

## In Scope

- Claudeチャット画面の実務利用
- プロンプトパターンと思考モデル
- 文書要約、文章作成、計画、調査支援
- プロジェクト、Artifacts、ファイル分析、多言語ライティングの概念的利用
- Claude Codeを任意の上級ルートとして紹介

## Out of Scope

- AIモデル構築やファインチューニング
- API連携やSDKチュートリアル
- 他AIツールとの詳細比較
- 法律・医療・金融判断の代替
- 会社固有の非公開ポリシーの詳細解釈

## Course Tone

Clear, practical, respectful to adult beginners, and focused on real work. Avoid childish examples, unexplained jargon, and developer-first assumptions.
