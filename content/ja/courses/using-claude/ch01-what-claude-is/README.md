# Claudeとは何か、何ではないか

この文書はGitBookでClaudeコースを探索し執筆するための基準です。

## 目標

Claudeの機能と限界を理解します。

## 学習成果

Claudeを文章作成、思考、調査、文書、コーディング支援のアシスタントとして説明できます。

## 重要概念

権威ではなく限界を持つAIアシスタントとしてのClaude。

## 実務での使い方

Claudeを初めて使う同僚に説明する。

## 安全メモ

機能の利用可否はClaudeのプランや現在の画面によって変わる場合があります。自分のClaude画面で確認してください。

実務で使う前に、人が事実、文脈、表現、機密情報の有無を確認する必要があります。
