# Chapter Structure

Use this chapter order when adding lessons.

1. Basic digital literacy
2. AI fundamentals
3. Prompting and AI collaboration
4. Data and document handling
5. Automation and workflow design
6. Practical mini-projects
7. Real-world application patterns

## Authoring Rule

One lesson should teach one major concept. If a topic is too broad, split it into smaller lessons.

Each chapter builds on the previous chapter and should include a simple exercise or thinking question before moving forward.
