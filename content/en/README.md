# Tech Bridge Book

Tech Bridge Book is a GitBook document collection for non-developers who want to learn AI and practical technology from beginner basics to real work use.

This first structure is not a full lesson set. It defines the learning path, language folders, terminology practice, safety baseline, and authoring templates before lesson writing begins.
