# Learner Guide

This book is designed for slow reading and immediate practice.

## Suggested Method

- Check the [glossary](glossary.md) when a term is unfamiliar.
- Translate examples into a work situation you know.
- Use fictional data instead of sensitive real data.
- Check AI output before trusting it.

## What You Need

- A web browser
- A document or note tool
- A spreadsheet tool is helpful
- An AI tool only when a lesson asks for it

## Learning Goal

The goal is not to become a developer. The goal is to understand technology, ask better questions, and judge results safely in practical work.
