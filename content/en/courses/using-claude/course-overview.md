# Course Overview: Using Claude

## Course Purpose

This course is not a Claude feature manual. It helps learners build habits for thinking, asking, verifying, and improving work with Claude.

## Learning Outcomes

- Explain what Claude is and how chatbot, writing partner, thinking partner, research assistant, document assistant, coding helper, and Claude Code usage differ.
- Choose practical use cases such as summarizing documents, improving emails, organizing meeting notes, drafting proposals, and creating tables or checklists.
- Recognize when not to use Claude and when human verification is required.
- Write clear prompts that include context, goal, format, and constraints.
- Review Claude output for accuracy, safety, and quality before turning it into final work.

## Prerequisite Knowledge

- Can use a web browser and type comfortably.
- Has access to Claude or a Claude-like interface approved by their organization.
- No programming knowledge is required for the main course.
- Claude Code appears only as an optional advanced path.

## In Scope

- Practical use of Claude chat interfaces
- Prompt patterns and mental models
- Document summarization, writing, planning, and research support
- Conceptual use of projects, artifacts, file analysis, and multilingual writing
- Claude Code as an optional advanced path

## Out of Scope

- Building or fine-tuning AI models
- API integration and SDK tutorials
- Detailed comparisons with other AI tools
- Replacing legal, medical, or financial judgment
- Interpreting private company policy in detail

## Course Tone

Clear, practical, respectful to adult beginners, and focused on real work. Avoid childish examples, unexplained jargon, and developer-first assumptions.

## Multilingual Rule

Korean is the canonical source. English and Japanese documents preserve the same structure and meaning with natural localization.
