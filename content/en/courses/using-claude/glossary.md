# Claude Course Glossary

Korean is the canonical terminology source. English and Japanese versions mirror the same term list with localized explanations.

| Term | Language | Definition |
|---|---|---|
| AI assistant | English | Software that responds to instructions and helps with thinking, writing, analysis, or planning. |
| Claude | English | Anthropic's AI assistant for chat, writing, document analysis, coding support, and structured thinking. |
| Prompt | English | The instruction or question you give Claude. |
| Context | English | Background information Claude needs to answer well. |
| Hallucination | English | A confident-sounding answer that may be wrong or unsupported. |
| Verification | English | Checking Claude output against reliable sources or human knowledge. |
| Artifact | English | A Claude output workspace for documents, code, tables, or interactive drafts when available. |
| Project | English | A Claude feature for grouping related chats, context, and instructions when available. |
| Session | English | One conversation or work period with Claude. |
| Token | English | A small unit of text processed by an AI model. |
| Output | English | The answer or content Claude produces. |
| Input | English | The text, file, or instruction provided to Claude. |
| Workflow | English | A repeatable sequence of steps for completing work. |
| Safety checklist | English | A short review list for privacy, accuracy, and responsible use. |
| Localization | English | Adapting meaning, tone, and examples for another language or culture. |
