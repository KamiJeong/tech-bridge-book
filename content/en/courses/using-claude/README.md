# Using Claude

Think, ask, verify, and refine: practical AI collaboration for real work

This course is not a Claude feature manual. It helps learners build habits for thinking, asking, verifying, and improving work with Claude.

## Target Audience

non-developers, beginner developers, planners, product managers, founders, marketers, operators, designers, researchers, customer support staff, and QA staff.

## What Learners Will Be Able To Do

- Explain what Claude is and how chatbot, writing partner, thinking partner, research assistant, document assistant, coding helper, and Claude Code usage differ.
- Choose practical use cases such as summarizing documents, improving emails, organizing meeting notes, drafting proposals, and creating tables or checklists.
- Recognize when not to use Claude and when human verification is required.
- Write clear prompts that include context, goal, format, and constraints.
- Review Claude output for accuracy, safety, and quality before turning it into final work.

## Safety Baseline

Before using output in real work, a human must review facts, context, tone, and sensitive information.
