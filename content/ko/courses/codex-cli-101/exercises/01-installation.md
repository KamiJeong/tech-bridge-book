# 실습 01: 설치 전 안전 점검

## 목표

Codex CLI를 실행하기 전에 실습 환경이 안전한지 점검합니다.

## 준비

- 공식 설치 안내
- 실습용 저장소
- 로그인에 사용할 계정 정보

## 문제

Codex 실습에 사용해도 되는 저장소인지 판단하는 체크리스트를 만듭니다.

## 진행 방법

1. 저장소에 비밀 정보가 있는지 직접 확인합니다.
2. Codex에게 "실습 전 안전 체크리스트"를 만들게 합니다.
3. 체크리스트에서 내 상황에 맞지 않는 항목을 수정합니다.
4. 공식 설치 안내와 오래된 블로그 글의 차이를 비교합니다.

## 제출물

- 실습 가능 여부 판단
- 제외해야 할 파일 유형 3개
- 현재 공식 안내에서 확인한 설치 요구사항

## 확인 질문

- 민감한 저장소를 실습 대상으로 삼지 않았나요?
- 설치 방법을 현재 공식 안내로 확인했나요?
