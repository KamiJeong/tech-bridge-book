# 실습 03: 낯선 코드베이스 읽기

## 목표

Codex와 함께 프로젝트 구조, 실행 시작점, 테스트 위치를 찾습니다.

## 준비

- 처음 보는 공개 저장소 또는 팀에서 공개 가능한 예제 저장소
- 메모할 문서

## 문제

새 프로젝트를 맡았다고 가정하고 30분 안에 "첫 번째 코드 읽기 지도"를 만듭니다.

## 진행 방법

1. Codex에게 주요 폴더를 설명하게 합니다.
2. 실행 시작점 후보를 찾게 합니다.
3. 테스트 위치와 실행 명령 후보를 찾게 합니다.
4. Codex 설명과 실제 파일을 비교합니다.

## 제출물

- 프로젝트 목적
- 주요 폴더 5개와 역할
- 먼저 읽을 파일 5개
- 아직 확인이 필요한 질문 3개

## 확인 질문

- 실제 파일을 열어 Codex 설명을 확인했나요?
- 모르는 부분을 작은 질문으로 나누었나요?
