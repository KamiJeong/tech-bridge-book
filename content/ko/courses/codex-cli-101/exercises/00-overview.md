# 실습 00: Codex CLI 개념 확인

## 목표

Codex CLI를 챗봇이 아니라 프로젝트 폴더 안에서 일하는 AI 코딩 에이전트로
이해합니다.

## 준비

- 실습용 저장소 하나
- 터미널
- `git status`를 실행해도 되는 안전한 폴더

## 문제

Codex에게 저장소 구조를 설명하게 한 뒤, 설명이 실제 파일과 맞는지 확인합니다.

## 진행 방법

1. 저장소 루트에서 Codex를 엽니다.
2. "아직 수정하지 말고 저장소 구조만 설명해줘"라고 요청합니다.
3. Codex가 말한 주요 파일 3개를 직접 열어 봅니다.
4. 맞는 설명과 애매한 설명을 나누어 적습니다.

## 제출물

- 저장소 목적 한 문장
- 먼저 읽을 파일 3개
- Codex가 추측한 부분 1개

## 확인 질문

- Codex가 확인한 사실과 추측을 구분했나요?
- 파일 수정 없이 분석만 진행했나요?
