# Claude 안전 체크리스트

Use this checklist before sharing information with Claude and before using Claude output in real work.

## Checklist

- Claude 답변을 그대로 믿지 말고 중요한 내용은 확인한다.
- 비밀번호, API 키, 인증 정보, 접속 토큰을 붙여 넣지 않는다.
- 고객 개인정보, 주민등록번호, 연락처, 결제 정보 등 민감한 개인 데이터를 공유하지 않는다.
- 회사 기밀, 내부 전략, 비공개 문서를 조직 정책 없이 공유하지 않는다.
- 법률, 의료, 금융, 안전 관련 결정의 유일한 근거로 사용하지 않는다.
- 최신 사실이라고 가정하지 말고 날짜와 출처를 확인한다.
- 사람의 검토 없이 최종 업무 결과물로 제출하지 않는다.
- 생각을 피하기 위해 쓰지 말고 더 잘 생각하기 위해 사용한다.
- 검증할 수 없는 업무에는 사용 범위를 줄이거나 사용하지 않는다.
- 파일 업로드 전 권한, 보관 정책, 공유 가능 범위를 확인한다.

## File Upload Rule

기능 제공 여부는 Claude 요금제와 현재 인터페이스에 따라 달라질 수 있습니다. 항상 본인의 Claude 화면에서 확인하세요.

공식 업무에 사용하기 전에는 사람이 사실, 맥락, 표현, 민감정보 포함 여부를 검토해야 합니다.
