# Course Overview: Claude 사용하기

## Course Purpose

이 과정은 Claude 기능 설명서가 아닙니다. 학습자가 Claude와 함께 생각하고, 질문하고, 검증하고, 결과를 개선하는 습관을 익히도록 돕습니다.

## Learning Outcomes

- Claude가 무엇인지와 챗봇, 글쓰기 파트너, 사고 파트너, 조사 보조자, 문서 보조자, 코딩 도우미의 차이를 설명한다.
- 긴 문서 요약, 이메일 개선, 회의록 정리, 제안서 초안, 표와 체크리스트 생성 등 실무 활용 사례를 고른다.
- Claude를 쓰면 안 되는 상황과 검증이 필요한 상황을 구분한다.
- 맥락, 목표, 형식, 제약을 포함한 명확한 프롬프트를 작성한다.
- Claude 답변을 사실, 안전, 품질 기준으로 검토하고 최종 산출물로 다듬는다.

## Prerequisite Knowledge

- Can use a web browser and type comfortably.
- Has access to Claude or a Claude-like interface approved by their organization.
- No programming knowledge is required for the main course.
- Claude Code appears only as an optional advanced path.

## In Scope

- Claude 채팅 인터페이스의 실무 사용
- 프롬프트 패턴과 사고 모델
- 문서 요약, 글쓰기, 계획, 조사 지원
- 프로젝트, 아티팩트, 파일 분석, 다국어 글쓰기의 개념적 사용
- Claude Code를 선택형 고급 경로로 소개

## Out of Scope

- AI 모델 구축 또는 파인튜닝
- API 통합과 SDK 사용법
- 다른 AI 도구의 상세 비교
- 법률, 의료, 금융 판단의 대체
- 회사별 비공개 정책 해석

## Course Tone

Clear, practical, respectful to adult beginners, and focused on real work. Avoid childish examples, unexplained jargon, and developer-first assumptions.

## Multilingual Rule

한국어가 원문 기준입니다. 영어와 일본어 문서는 같은 구조와 의미를 자연스럽게 옮깁니다.
