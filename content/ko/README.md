# Tech Bridge Book

Tech Bridge Book은 비개발자가 AI와 실무 기술을 기초부터 업무 활용까지 배울 수 있도록 돕는 GitBook 문서입니다.

이 첫 구조는 전체 수업을 작성하기 전 단계입니다. 지금은 학습 흐름, 언어별 폴더, 용어 관리 방식, 안전 기준, 작성 템플릿을 먼저 정리합니다.

## 읽기 전 사전 준비

이 책은 비개발자 학습자를 위해 만들어졌습니다. 빠른 정답보다는 스스로 판단하는 힘을 기르는 것을 목표로 합니다. 한 장(章)을 마치는 데에는 보통 10~20분의 집중 시간이 필요합니다.

- 웹 브라우저, 문서 편집기, 스프레드시트의 기본 사용에 익숙해야 합니다.
- 일터에서 마주치는 문서, 보고, 일정, 고객 응대 같은 현실 상황을 떠올릴 수 있어야 합니다.
- AI 도구에 회사 기밀, 고객 정보, 개인정보를 그대로 붙여 넣지 않습니다. 자세한 기준은 [AI 안전과 개인정보](ai-safety-privacy.md)를 먼저 읽어 주세요.
- 정답을 외우려 하지 말고, 왜 그렇게 작동하는지 묻는 습관을 가져 주세요.

준비가 되었다면 [소개](introduction.md) 페이지로 이동해 학습을 시작합니다.
