# Tech Bridge Book

Tech Bridge Book은 비개발자가 AI와 실무 기술을 기초부터 업무 활용까지 배울 수 있도록 돕는 GitBook 문서입니다.

이 첫 구조는 전체 수업을 작성하기 전 단계입니다. 지금은 학습 흐름, 언어별 폴더, 용어 관리 방식, 안전 기준, 작성 템플릿을 먼저 정리합니다.

## 시작 순서

1. [소개](introduction.md)
2. [커리큘럼 개요](curriculum-overview.md)
3. [학습자 가이드](learner-guide.md)
4. [AI 안전과 개인정보](ai-safety-privacy.md)
5. [장 구조](chapters/README.md)

## 작성자를 위한 기준

새 수업은 [템플릿](templates/README.md)을 사용합니다. 한국어가 기준 원문이며, 영어와 일본어는 의미와 구조를 맞춰 자연스럽게 현지화합니다.
